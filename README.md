# Krejčovství Alibaba

Professional website for Alibaba Tailoring Shop.

## About

This is a modern, responsive website for Krejčovství Alibaba (Alibaba Tailoring Shop), featuring:

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Czech Language**: Content tailored for Czech-speaking customers
- **Service Showcase**: Highlights six main tailoring services
- **Contact Information**: Complete business details including address, hours, and contact info
- **Modern UI**: Clean, professional design with gradient effects and smooth animations

## Quick Start

Simply open `index.html` in a web browser to view the website locally.

For local development with live preview:

```bash
# Using Python 3
python3 -m http.server 8000

# Then open http://localhost:8000 in your browser
```

## Services Offered

1. **Šití na míru** - Custom tailoring
2. **Opravy oděvů** - Clothing repairs
3. **Zkrácení a prodloužení** - Hemming and lengthening
4. **Úpravy velikostí** - Size alterations
5. **Svatební šaty** - Wedding dresses
6. **Expresní služby** - Express services

## Technology

- Pure HTML5
- CSS3 with responsive design
- No frameworks or build tools required
